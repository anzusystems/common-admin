# embedExternalImageInline

## Features
- only for migration, not used by editor
- replaced by `embedImageInline` after media migration

## Node schema

```json
{
  "name": "embedExternalImageInline",
  "groups": [
    "inline"
  ],
  "attrs": {
    "id": {
      "default": ""
    },
  }
}
```

## Node JSON example

```json
{
  "type": "embedExternalImageInline",
  "attrs": {
    "id": 23
  }
}
```
