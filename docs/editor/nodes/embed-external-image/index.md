# embedExternalImage

## Features
- only for migration, not used by editor
- replaced by `embedImage` after media migration

## Node schema

```json
{
  "name": "embedExternalImage",
  "groups": [
    "embed"
  ],
  "attrs": {
    "id": {
      "default": ""
    }
  }
}
```

## Node JSON example

```json
{
  "type": "embedExternalImage",
  "attrs": {
    "id": 23
  }
}
```
