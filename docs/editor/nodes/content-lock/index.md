# contentLock

## Features
- User can add this separator to separate multiple 2 parts of document - free and paid part
- Only one per article, callback to set article field.

## Node schema

```json
{
  "name": "contentLock",
  "groups": [
    "lock"
  ]
}
```

## Node JSON example

```json
{
  "type": "contentLock"
}
```
